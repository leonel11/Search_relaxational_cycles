Формулировка задачи:

Рассматривается математическая модель нейронной системы, реализующая релаксационные автоколебания в многомерном пространстве. 

С помощью входных параметров определяются вид дифференциального уравнения и период колебаний 
(колебания начинаются в начальный момент времени t = 0 из состояния покоя - точки -0). 

Требуется найти координаты конечной точки sln (y).
(подробнее о релаксационных автоколебаниях можно прочитать из статей DU0155.pdf, DU0919.pdf, DU1675.pdf)

Входные параметры:
	    dim (m) - размерность пространства для задачи;
	    start_point - начальная точка, для которой происходят колебания;
	    d - множитель, на который домножается правая часть дифференциального уравнения;
	    alpha, beta - параметры, определяющие период (T0), до которого необходимо рассматривать колебания в системе;
	    eps - точность цикла (если 2 точки в последовательности решений отличаются друг от друга не более чем на eps по модулю, 
		      то последняя полученная точка считается стационарной;
	    h - малый параметр, определяющий величину, на которую изменяется время t на каждом шаге 
		    (необходимо для метода Рунге-Кутты).
 
 Начальные условия задачи: 
	    dim >= 1;
	    d >= 0;
	    alpha, beta > 0
	    alpha-beta-1 > 0;
	    0 <= t <= T0, где t - время, в течение которого происходят колебания в системе;
	    T0 = alpha + 1 + (beta+1)/(alpha-beta-1);
